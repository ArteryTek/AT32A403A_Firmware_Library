  this demo is based on the at-start board, in this demo, pc6-9 toggle each
  compare time.
  cc1 overflow rate = tmr2 counter clock / c1dt_val = 152.59 hz
  cc2 overflow rate = tmr2 counter clock / c2dt_val = 228.87 hz
  cc3 overflow rate = tmr2 counter clock / c3dt_val = 457.76 hz
  cc4 overflow rate = tmr2 counter clock / c4dt_val = 915.67 hz 

  for more detailed information. please refer to the application note document AN0085.
