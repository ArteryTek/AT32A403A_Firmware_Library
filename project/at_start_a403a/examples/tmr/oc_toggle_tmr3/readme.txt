  this demo is based on the at-start board, in this demo, pa6 pa7 pb0 pb1
  output waveform.
  cc1 update rate = tmr3 counter clock / ccr1_val = 305 hz
  cc2 update rate = tmr3 counter clock / ccr2_val = 610 hz
  cc3 update rate = tmr3 counter clock / ccr3_val = 1220 hz
  cc4 update rate = tmr3 counter clock / ccr4_val = 2440 hz

  for more detailed information. please refer to the application note document AN0085.
